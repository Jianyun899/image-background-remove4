# Image Background Remover — MVP 需求文档

**版本：** v0.1
**日期：** 2026-03-24
**状态：** 草稿

---

## 1. 产品概述

一个在线图片背景去除工具，用户上传图片后自动去除背景，下载透明 PNG。目标是快速上线验证市场，技术栈轻量，零运维成本。

---

## 2. 目标用户

- 电商卖家（产品图去背景）
- 设计师（快速抠图）
- 普通用户（证件照、头像处理）

---

## 3. MVP 功能范围

### ✅ 包含

| 功能 | 描述 |
|------|------|
| 图片上传 | 支持拖拽 / 点击上传，格式：JPG、PNG、WEBP |
| 背景去除 | 调用 Remove.bg API 自动处理 |
| 结果预览 | 原图 vs 去背景图对比展示 |
| 下载 | 下载透明背景 PNG |
| 隐私保护 | 图片仅在内存中处理，不落盘不存储 |

### ❌ 不包含（MVP 阶段暂不做）

- 用户登录 / 账号系统
- 使用次数限制 / 付费墙
- 批量处理
- 替换背景颜色 / 图片
- 历史记录

---

## 4. 技术架构

```
用户浏览器
    │
    ▼
Cloudflare Pages（静态前端）
    │  上传图片（multipart）
    ▼
Cloudflare Worker（后端逻辑）
    │  转发图片
    ▼
Remove.bg API
    │  返回透明 PNG
    ▼
Cloudflare Worker → 直接返回给浏览器
（无存储，纯内存流转）
```

**技术选型：**

| 层级 | 技术 |
|------|------|
| 前端 | 纯 HTML + CSS + JS（单文件，无框架） |
| 后端 | Cloudflare Workers |
| 图片处理 API | Remove.bg API |
| 部署 | Cloudflare Pages + Workers |
| 存储 | 无 |

## 6. 关键约束

| 项目 | 限制 |
|------|------|
| 图片大小 | 最大 5MB（Remove.bg 免费版限制） |
| 图片格式 | JPG / PNG / WEBP |
| 并发 | 依赖 Remove.bg API 配额（免费版 50次/月） |
| 隐私 | 图片不存储，处理完即丢弃 |

---

## 7. Remove.bg API 集成

- **接口：** `POST https://api.remove.bg/v1.0/removebg`
- **认证：** Header `X-Api-Key: YOUR_API_KEY`
- **输入：** multipart `image_file`
- **输出：** 透明背景 PNG 二进制
- **API Key 存放：** Cloudflare Worker 环境变量（不暴露给前端）
